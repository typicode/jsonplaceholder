Online Fake REST API for Developers

powered by JSON Server

What?

JSONPlaceholder is a free online REST service you can use whenever you need some fake data.

Example

Run this code in a console or from anywhere (CORS and JSONP supported)

var root = 'http://jsonplaceholder.typicode.com';

$.ajax({
  url: root + '/posts/1',
  method: 'GET'
}).then(function(data) {
  console.log(data);
});
